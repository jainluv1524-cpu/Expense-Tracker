# Smart Expense Tracker

Modern single-page expense tracker web app with login, animated background, interactive charts and local storage persistence.

**No backend, no build step** — just open `index.html` in the browser.

## Features

- Secure browser-based login (password stored in localStorage)
- Add / delete income & expense entries
- Real-time running balance calculation
- Confetti explosion when adding income
- Interactive doughnut chart for expense categories (Chart.js)
- Responsive layout
- All data saved in browser (persists after refresh/close)

## Technologies

- HTML5 + CSS3 (animations, variables, flex/grid)
- Vanilla JavaScript (ES6+)
- Chart.js (via CDN)
- Font Awesome icons (via CDN)

## How to Run

1. Clone or download this repository
2. Open `index.html` in any modern browser (Chrome, Edge, Firefox)

**Default password**: `password123`  

## Deployment (free hosting)

### GitHub Pages (recommended)

1. Go to repository → **Settings** → **Pages**
2. Source → Deploy from a branch → main → / (root)
3. Save → wait 1–2 minutes
4. Your app will be live at:  
   https://**yourusername**.github.io/**smart-expense-tracker**/
